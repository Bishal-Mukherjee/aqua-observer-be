# 🐬 App Backend — Freshwater Species Sighting System

This is the backend service that powers the mobile app used by volunteers to record sightings of freshwater species.

---

## 📦 Tech Stack

- **Node.js**
- **Express.js + Typescript**
- **PostgreSQL**
- **Supabase**
